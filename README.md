# Flutter Todo List

A simple, clean todo list app built with Flutter and Material .

## Features

- Add and delete tasks
- Mark tasks as complete with line crossing through the task.
- Task counter showing total and completed
- Clean card-based UI with a purple theme

## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- An emulator or physical device

### Run the app

```bash
flutter run
```
